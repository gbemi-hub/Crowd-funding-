# Crowdfunding Product Page

A responsive and interactive crowdfunding product page built as part of a **Frontend Mentor challenge**. This project allows users to explore a campaign, select reward tiers, pledge support, and track funding progress.

## Overview

The page simulates a real crowdfunding experience. Users can view detailed information about the project, choose from different reward options, and receive visual feedback through modal confirmations and dynamic updates.

## Features

* Responsive design for all screen sizes
* Interactive pledge selection modal
* Real-time update of total funds and backers
* Bookmark toggle functionality
* Custom progress bar
* Clean, modern user interface

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Vanilla JavaScript
